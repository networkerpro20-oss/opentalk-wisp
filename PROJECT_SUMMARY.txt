📝 COMANDOS QUICK START
═══════════════════════════════════════════════════════════════════════════════

   # Setup inicial
   pnpm install
   cd apps/backend && pnpm prisma generate
   pnpm prisma migrate dev

   # Desarrollo
   pnpm dev                    # Backend + Frontend
   pnpm dev:backend            # Solo backend
   pnpm dev:frontend           # Solo frontend

   # Database
   pnpm prisma studio          # GUI database
   pnpm prisma migrate dev     # Nueva migración

   # Build
   pnpm build                  # Build todo
   pnpm build:backend          # Build backend
   pnpm build:frontend         # Build frontend

   # Deployment
   ./deploy.sh                 # Script interactivo
   docker-compose up -d        # Docker

   # Helpers
   ./commands.sh               # Ver todos los comandos

🐛 TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════════

   Error: Cannot find '@prisma/client'
   → cd apps/backend && pnpm prisma generate

   Error: Connection refused
   → Verificar backend en puerto 3001
   → Revisar NEXT_PUBLIC_API_URL

   Error: WhatsApp no conecta
   → QR expira en 60s, recargar
   → Solo una instancia por número

   Error: CORS blocked
   → Agregar URL en main.ts
   → Verificar FRONTEND_URL

   Para más ayuda:
   → ./commands.sh (opción 3: Troubleshooting)
